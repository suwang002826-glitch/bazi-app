# Codex 项目规则

本项目是中文八字排盘 App。Codex 在本仓库内工作时必须遵守这些规则。

## 协作规则

- 所有用户界面文字必须是简体中文。
- 不要直接修改 `main` 分支；每个功能都应使用独立任务分支。
- 每次只处理当前任务，不顺手重构无关代码。
- 修改排盘逻辑时，必须同步检查 `排盘规则.md` 和 `验收样例.md`。
- 遇到命理流派差异，先记录差异，不自行决定为唯一正确答案。
- 提交或合并前必须运行构建检查。

## 产品边界

- 第一版只做排盘，不做 AI 断命。
- 第一版不做登录、支付、会员、社区。
- 手机端优先，页面必须适合手指点击。
- 页面不允许出现英文按钮、英文提示或英文标题。

## 技术约定

- 使用 React + TypeScript + Vite。
- 优先使用成熟历法库，不手写复杂历法算法。
- 当前排盘库候选为 `lunar-typescript`。
- 本地保存优先使用浏览器 `localStorage`。
- 复杂规则必须先写测试样例，再改实现。
