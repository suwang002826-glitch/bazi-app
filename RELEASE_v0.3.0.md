## 二、v0.3.0新增功能详情
### 1. 流年排盘
✅ **规则完全对齐原局**：以立春为岁首，直接复用`getYearPillar`函数计算流年干支，和原局年柱逻辑100%一致，无口径差异
✅ **默认覆盖范围**：出生年（0岁）到80岁，共81个流年
✅ **每个流年包含字段**：
  - 公历年、对应年龄
  - 干支、纳音、对日主的十神
  - 立春日期、流年起止日期（立春到下一年立春前）
  - 该年12个流月完整数据

### 2. 流月排盘
✅ **规则完全对齐原局**：以十二节为月首，直接复用`getMonthPillar`函数+五虎遁法计算流月干支，和原局月柱逻辑100%一致，无口径差异
✅ **覆盖范围**：每个流年包含12个流月（寅月到丑月）
✅ **每个流月包含字段**：
  - 月份名称（寅月/卯月...丑月）、对应节令名称
  - 干支、纳音、对日主的十神
  - 流月起止日期（节令到下一节令前）

### 3. 兼容性
✅ 新增`liuNian`字段到排盘返回结果，不修改原有任何字段，完全兼容旧版本调用
✅ 无额外参数要求，调用`buildBaziChart`自动返回流年流月数据，无需新增入参

## 四、调用方式
流年流月结果已自动集成到`buildBaziChart`返回值中，无需额外参数：
```javascript
const result = buildBaziChart({
  birthDate: '2000-12-17',
  birthTime: '10:00:00',
  gender: '男',
  longitude: 120
});

// 流年结果在result.liuNian中，数组格式，按年龄从小到大排序
console.log(result.liuNian[0].ganZhi); // 出生年干支：庚辰
console.log(result.liuNian[24].ganZhi); // 24岁流年干支：甲辰
console.log(result.liuNian[24].months[0].ganZhi); // 24岁正月干支：丙寅
```
