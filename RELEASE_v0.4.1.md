# 八字排盘小程序 v0.4.1 发布说明

发布日期：2026-07-04

## 版本定位

v0.4.1 是「大运流年可视化 + 本地命例历史 + 基础命理说明 + 神煞样本入口」的稳定整理版。

核心宗旨不变：精准、稳定、专业。所有新增能力都不得破坏四柱、大运、流年、流月和农历 data-pack 的既有验收结果。

## 已完成功能

### 1. 大运流年结果页增强

- 结果页「大运流年」页签支持大运切换、流年切换、流月展开。
- 页面使用后端输出的真实 `daYun`、`liuNian`、`liuYue` 数据，不再使用假数据。
- 新增结果页辅助模块，支持默认折叠、当前流年定位、流年/流月格式化。

### 2. 本地命例历史

- 支持在结果页保存命例。
- 支持首页展示历史命例、点击恢复、删除。
- 该能力使用小程序本地缓存，不依赖远程后端。

### 3. 城市选择与真太阳时辅助

- 新增城市选择辅助模块。
- 扩充主要城市经纬度数据，覆盖北京、上海、广州、深圳、乌鲁木齐、哈尔滨、拉萨等常用城市。
- 输入页选择城市后可自动填充出生地、经度、纬度和坐标来源。

### 4. 字段解释

- 新增 `explanations.js`，提供十神、藏干、空亡、纳音、神煞、十神旺衰等字段说明。
- 说明只做标准信息解释，不做主观断语。

### 5. 问真神煞样本入口

- 新增 `shenshaSamples.js`。
- 新增 `docs/问真神煞样本采集指南.md`。
- 新增 `npm run sample:shensha`。
- 当前只建立样本采集和结构校验入口，不实现神煞算法。
