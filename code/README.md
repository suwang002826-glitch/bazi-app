# 玄问 AI 微信小程序测试版

这是一个 AI 赋能中国道教传统文化的小程序测试版，当前包含首页道教日课、八字排盘、六爻起卦、奇门起局、我的档案和命例档案。当前版本不包含会员页，也不恢复 `pages/member`。

## 当前功能

- 首页：青玄万年历、今日道历口径、斋忌与修持建议、八字/六爻/奇门入口、命例档案入口、最近本地记录。
- 八字：小程序端负责出生信息录入、后端接口调用和结果展示；排盘算法、农历转换、节气、真太阳时、权威来源和验收边界由后端负责。
- 六爻：手动/时间/数字起卦、本卦变卦、世应、六亲、六神、用神强弱、动爻优先级、回头生克、伏神飞神提示、应期与风险提示。
- 奇门：问事分类、起局时间、节气三元、阴阳遁、局数、九宫盘、值符值使、用宫、三吉门、慎用门、门迫、伏吟、反吟与行动建议。
- 命例档案：八字/六爻/奇门分类存档，支持应验时间、准确点、偏差点、现实反馈和验证状态，用于内测复盘准确度。

## UI 方向

- 视觉基调改为黑蓝、克制、专业，少量暖金只做道教器物感点睛。
- 首页去掉搜索、产品说明、横幅、万年历和最近解读，仅展示八字、六爻、奇门三大产品入口。
- 首页已接入生成式传统视觉资产：新中式道教 logo、八字图、六爻图、奇门图，并全部压缩到 200KB 以下。
- 背景包含低透明度的太极、八卦、北斗七星、日月和如意纹，保证有传统气质但不抢信息。
- 标题保留楷书气质，正文使用更清爽的系统中文字体，避免大段阅读疲劳。
- 结果页先给小白可读的结论摘要、白话解释和修身建议，再展开专业细盘。
- 已将重复底纹抽成 `components/x-pattern` 自定义组件，并在 `app.json` 通过 `usingComponents` 全局声明，同时启用 `lazyCodeLoading: "requiredComponents"`。
- 代码包内图片、音频等资源应保持小体积；当前导入包不内置图片/音频资源，预览物料不进入代码包。

## 手机微信试用

1. 先启动后端服务：`node backend/server.js`。
2. 打开微信开发者工具，选择“导入项目”。
3. 项目目录选择本仓库的 `code` 目录。
4. 填写可开发的小程序 AppID；没有 AppID 时可用测试号调试，但真机预览和体验版能力会受限制。
5. 点击“编译”确认首页、八字、六爻、奇门、我的五个底部页面都可打开。
6. 点击八字页“立即排盘”，确认能访问 `http://127.0.0.1:8787/bazi/calculate`。
7. 点击“预览”，用绑定开发者账号的手机微信扫码。
8. 若要给小范围用户试用，在微信公众平台上传版本、设为体验版，并添加体验成员。

## 测试边界

当前八字测试入口不再把小程序端作为算法真相源。若后端服务不可用，八字页应提示服务不可用，而不是退回本地计算。农历、闰月、节气和真太阳时继续按后端准入数据与验收脚本推进。所有结论均为传统文化学习、趋吉避凶与产品验证参考，不作绝对化断语，也不用于医疗、法律、投资等重大决策。
