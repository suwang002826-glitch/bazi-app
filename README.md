# 八字排盘 App

这是一个面向零编程基础协作的八字排盘 App 起步项目。第一版目标是做出一个全中文、手机可用、可保存记录、可持续校验排盘准确性的工具。

## 当前状态

- 已创建 React + TypeScript + Vite 项目。
- 已接入 `lunar-typescript` 作为第一版排盘计算候选库。
- 已完成中文输入界面、四柱展示、藏干、五行、十神、纳音、旬空和本地保存记录。
- 已补充 GitHub 协作规则、Issue 模板、Pull Request 模板和验收样例。

## 本地运行

```bash
npm install
npm run dev
```

## 项目规则

- 所有用户界面必须使用简体中文。
- 第一版只做排盘，不做 AI 断命、付费、登录、社区。
- 每个任务必须先写 Issue，再开任务分支，再通过 Pull Request 合并。
- `main` 分支保持稳定可运行，不直接修改。
- 排盘结果必须用 `验收样例.md` 持续复核。
- 遇到流派差异，先更新 `排盘规则.md`，不要直接改算法定论。

## 两人分工

- 产品负责人：界面、文案、流程、手机端体验、用户反馈。
- 规则负责人：排盘规则、样例对照、准确性验收、流派差异记录。
- Codex：写代码、改 bug、补测试、解释改动、维护项目结构。
