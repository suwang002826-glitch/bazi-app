# 验收样例

这个文件用于记录排盘准确性样例。第一阶段目标是整理 20 个样例，后续扩展到 50 个以上。

当前批次：第一批基础样例。

样例状态：以下结果先作为计算参考和测试基准，后续需要由人工或第二来源复核。复核前不要把它们宣传为最终权威结果。

```md

```

## 第一批样例说明

- 这批样例覆盖普通公历、立春前后、子时、农历、农历闰月。
- 真太阳时默认关闭。
- 子时按 `docs/排盘规则.md` 的 `v0.1` 口径处理：第一版默认子时不换日。
- 大运起运暂不作为第一批通过标准。

- 男命顺排大运样例

- 已有命例档案中能复核现实反馈的样例
