# 验收样例

这个文件用于记录排盘准确性样例。第一阶段目标是整理 20 个样例，后续扩展到 50 个以上。

## 样例格式

```md
### 样例 001

- 来源：
- 性别：
- 历法：公历 / 农历
- 出生地：
- 出生时间：
- 是否使用真太阳时：
- 正确年柱：
- 正确月柱：
- 正确日柱：
- 正确时柱：
- 大运起运：
- 备注：
```

## 必备样例类型

- 普通公历生日样例
- 农历生日样例
- 农历闰月样例
- 立春前后样例
- 节气交接前后样例
- 23:00-00:59 子时样例
- 启用真太阳时前后会变化的样例
- 男命顺排大运样例
- 女命逆排大运样例
- 已有命例档案中能复核现实反馈的样例

- 样例用于 Pull Request 验收和回归检查。
