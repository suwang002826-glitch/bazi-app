# 月德贵人文字材料

材料类型：用户补充的口诀与解释文案。
用途：作为后续神煞点击说明、文案整理的参考材料。
限制：不作为新增命例，不改变算法就绪状态，不替代问真八字实际排盘样本。

## 口诀

寅午戌月生者见丙，申子辰月生者见壬，亥卯未月生者见甲，巳酉丑月生者见庚。

查法：以月支查四柱干支。

## 解释要点

- 月德贵人同天乙贵人一样，是传统说法中的吉星。
- 命局中逢月德贵人，传统说法中主一生处世无忧、化险为夷，平生少病，不犯官刑。
- 月德强调勤勉敏慧，即使命主身带此星，也需要自身勤勉自助，才容易在紧要关头获得帮助。
- 天月二德属于日月会合照临，传统说法中重视慈祥、仁厚、坦白无私。
- 二德以天德为重，月德次之；两者均作为解释材料收录，不能直接替代问真命例验收。
