# 驿马文字材料

材料类型：用户补充的口诀与解释文案  
用途：作为后续神煞点击说明、文案整理的参考材料  
限制：不作为新增命例，不改变算法就绪状态，不替代问真八字实际排盘样本

## 古诀

申子辰马在寅，寅午戌马在申，巳酉丑马在亥，亥卯未马在巳。

## 查法说明

以年支、日支查余三支。

## 解释要点

- 驿马传统说法中主走动、远行、出远门、迁移、奔波、换工作、出国等。
- 驿马坐旺地，传统说法中主利禄亨通。
- 驿马为喜用，传统说法中主动则有喜；四柱财官有力时称为真马。
- 驿马为喜用并自坐财官地，岁运逢财官星，传统解释多与升迁相关。
- 驿马为忌且逢冲，传统解释多与是非波动、奔波忙碌相关。
- 吉神坐马，多解释为乔迁之喜或顺动之利；凶神坐马，多解释为奔走四方、忙于生计。
- 驿马与财星、财官、贵人、正官、七杀、羊刃、劫煞、桃花、空亡等同柱或逢冲组合，均需后续结合问真实际命例再校验展示口径。
- 流年驿马逢冲，传统解释多与奔波、迁异职动、出国或远行相关。
