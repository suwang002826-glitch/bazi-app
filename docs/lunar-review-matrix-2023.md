这些样例通过自动校验后，只能证明草案数据内部一致。

完整农历排盘上线前仍必须补齐：

- 独立复核记录；
- `reviewStatus: "approved-for-runtime"`；
- 更多年份和完整农历年覆盖；
- 数据来源差异说明；
- 总验收通过记录。
