# 排盘规则

这个文件记录本项目采用的排盘规则。任何算法变化都必须先更新这里，再修改代码。

## 当前已采用

- 默认按输入时间直接排盘。
- 公历输入通过 `Solar.fromYmdHms(...).getLunar()` 转换。
- 农历输入通过 `Lunar.fromYmdHms(...)` 转换。
- 农历闰月暂按负数月份传入库函数。
- 四柱、藏干、五行、十神、纳音、旬空暂由 `lunar-typescript` 输出。

## 待确认

- 子时换日规则：晚子时是否归次日。
- 月柱规则：是否完全按节气换月。
- 真太阳时：是否启用，启用后如何根据出生地换算。
- 大运起运：使用哪一套起运算法。
- 农历闰月：需要用真实样例复核。
- 不同时区出生：是否支持海外出生地。

## 验收原则

- 每个争议规则至少准备 3 个对照样例。
- 样例来源必须记录清楚。
- 如果多个来源结果不同，先记录差异，不立即改代码。
- 只有规则负责人确认后，才修改排盘计算。
