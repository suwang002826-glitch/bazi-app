## 二、v0.2.0新增功能详情
### 1. 大运排盘核心逻辑
✅ **顺逆排规则**：阳年（甲丙戊庚壬）男命、阴年（乙丁己辛癸）女命顺排；阴年男命、阳年女命逆排，100%符合子平法规则
✅ **起运时间计算**：
  - 顺排：出生时间到下一个节的时间差，按「3天=1年、1天=4个月、1时辰=10天」折算
  - 逆排：出生时间到上一个节的时间差，同规则折算
  - 精确返回起运年龄（岁/月/天）、具体起运日期
✅ **大运干支列表**：默认排8步大运（覆盖0-80岁），每步管10年，包含：
  - 大运干支、纳音
  - 对应日主的十神
  - 起止年龄、起止公历年
✅ 无性别参数时大运返回null，兼容旧版本调用

### 2. 其他升级
✅ 前端农历选择器对接完成：小程序八字页支持农历/闰月选择，调用完整1901-2100农历数据包，移除“农历日期暂未覆盖”提示
✅ 默认加载1900-2100精确节气数据包，外部未传节气数据时不再使用近似算法，排盘精度稳定到秒级
✅ 临时垃圾文件已清理，无冗余测试脚本/乱码文件进入仓库

## 四、调用方式
大运结果已集成到`buildBaziChart`返回值中，传入`gender`参数（支持'男'/'女'/'male'/'female'/1/0）即可自动返回：
```javascript
const result = buildBaziChart({
  birthDate: '2000-12-17',
  birthTime: '10:00:00',
  gender: '男', // 新增性别参数，必填以生成大运
  longitude: 120
});

// 大运结果在result.daYun中
console.log(result.daYun.direction); // '顺'/'逆'
console.log(result.daYun.startAge); // 起运年龄
console.log(result.daYun.list); // 8步大运列表
```
